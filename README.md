# Al-Quran Web App

A beautiful, responsive, and modern Quran reader with Arabic text and English translation.

## Features

- Clean and attractive  UI
- Browse all 114 Surahs
- Read full Surah with Arabic + English Translation
- Global Search by English translation
- Settings Panel:
  - Arabic Font Selection (Amiri & Noto Naskh Arabic)
  - Adjustable Arabic & Translation font size
- Fully Responsive (Mobile + Desktop)
- Settings saved in localStorage
- Built with Shadcn/ui + Tailwind CSS

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn/ui
- **Language**: TypeScript
- **Fonts**: Amiri & Noto Naskh Arabic (Google Fonts)
